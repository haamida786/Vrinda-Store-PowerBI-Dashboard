# E-Commerce Sales Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard developed to analyze e-commerce sales performance across different categories, channels, states, genders, and monthly trends.

The dashboard transforms raw transactional data into actionable business insights through dynamic charts, KPI visuals, and interactive filtering features.

This project demonstrates practical skills in:

* Business Intelligence
* Data Visualization
* Sales Analytics
* Dashboard Development
* Power BI Reporting

---

# Objectives

The main objectives of this project are:

* Analyze sales amount across different product categories
* Compare sales performance across sales channels
* Study regional sales distribution by state
* Monitor monthly order and revenue trends
* Understand customer purchasing behavior based on gender
* Build an interactive business dashboard for decision-making

---

# Dashboard Features

The Power BI report contains multiple interactive visuals and analytical components.

## 1. Amount by Category

### Description

Displays total sales amount across different product categories.

### Insights

* Identifies top-performing categories
* Helps understand customer demand patterns
* Supports inventory and marketing decisions

---

## 2. Amount by Channel

### Description

Analyzes revenue generated through different sales channels.

### Possible Channels

* Online Platforms
* Mobile Applications
* Retail Sales
* Marketplace Channels

### Business Value

* Compare channel performance
* Identify high-revenue platforms
* Optimize sales strategies

---

## 3. Amount by Ship-State

### Description

Shows state-wise sales distribution.

### Benefits

* Identify high-performing regions
* Understand geographical demand
* Improve regional business planning

---

## 4. Order ID and Amount by Month

### Description

Tracks monthly sales and order trends.

### Insights

* Detect seasonal trends
* Analyze monthly business growth
* Monitor order fluctuations over time

---

## 5. Amount by Gender

### Description

Analyzes sales amount based on customer gender.

### Purpose

* Understand customer demographics
* Compare purchasing behavior
* Support targeted marketing campaigns
